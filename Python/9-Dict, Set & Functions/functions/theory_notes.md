# Day 06 — Functions in Python

## 1. What is a Function?

A function is a reusable block of code.

Example:
def greet():
    print("Hello")

---

## 2. Why Functions?

- Avoid repetition
- Improve readability
- Modular code

---

## 3. Function Syntax

def function_name(parameters):
    return value

---

## 4. Parameters vs Arguments

def add(a, b):  # parameters
    return a + b

add(2, 3)  # arguments

---

## 5. Return Statement

- Sends value back
- Ends function execution

---

## 6. Types of Functions

1. No return
2. With return
3. With parameters
4. Without parameters

---

## 7. Scope (Basic)

Local → inside function  
Global → outside function  

---

## 8. Best Practices

- Use meaningful names
- Keep functions small
- Avoid global variables

---

## Key Takeaways

- Functions = building blocks of applications
- Used everywhere (ML, backend, APIs)
