# Data Pipeline (End-to-End)

## 1. What is a Data Pipeline?

Flow:
Data Source → Processing → Storage → Analysis

---

## 2. Steps

1. Load data (CSV)
2. Clean data (Pandas)
3. Transform data
4. Store in database (SQL)
5. Query + analyze

---

## 3. Why Important?

- Used in ML pipelines
- Backend systems
- Analytics dashboards

---

## 4. Tools Used

- Pandas → processing
- SQLite → storage
- SQL → querying

---

## 5. Real-World Example

E-commerce:
CSV → Clean → DB → Dashboard

---

## Key Takeaways

- Pipeline = real system
- Data flows step-by-step
